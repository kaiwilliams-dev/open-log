Running in 60 seconds

1

Clone and install

Requires Bun and tmux. Both install in seconds via Homebrew.

$ git clone https://github.com/kaiwilliams-dev/open-log.git && cd open-log && bun install
2

Start the server

Reads your local Claude Code data from ~/.claude/projects/. Everything stays on your machine.

$ bun run start
3

Open and sync

Click Sync Now to pull your exact usage percentage from Anthropic. Takes about 5 seconds.

http://localhost:7777
