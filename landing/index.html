Everything you need to see

No cloud. No accounts. Just your local Claude Code data, beautifully displayed.

Real-time usage %

Exact session and weekly percentages synced directly from Anthropic. Know when to slow down before you hit the wall.

Cost tracking

See exactly how much you're spending per session. Token breakdown by type — input, output, cache creation, cache read.

Burn rate forecast

Your %/hour rate and a prediction of when you'll hit 100%. Plan your heavy coding sessions accordingly.

Skills & hooks

Which skills actually fire? How often? See trigger counts, 7-day trends, and expand for daily breakdowns.

Usage history

Monthly cost graphs with daily breakdown. See your spending patterns, peak days, and trends over time.

Activity heatmap

7-day hourly heatmap. Click any cell to see triggers, costs, skills, and hooks fired that hour.

Up and running in 60 seconds

1

Clone and install

Requires Bun and tmux. Both install in seconds.

$ git clone https://github.com/kaiwilliams-dev/open-log.git && cd open-log && bun install
2

Start the server

OpenLog reads your local Claude Code data from ~/.claude/projects/. Nothing leaves your machine.

$ bun run start
3

Open the dashboard

Navigate to localhost:7777 and click Sync Now. Your exact Anthropic usage appears in seconds.

http://openlog.local:7777
